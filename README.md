# wxJump
微信小游戏项目 跳一跳源码

# 食用方法
<div align=left><img width="350" height="400" src="https://github.com/xirtam-ch/ReadmeImages/blob/master/wxJump_config.jpg"/></div>

- 不要直接打开本项目
- 选择一个空目录，使用微信小程序开发工具 创建一个小游戏项目（不能使用小程序项目）
- 将本项目中的文件覆盖到创建好的项目中
- 开始食用
